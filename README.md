## Project Setup

```sh
git clone https://github.com/japhg/laravel-api.git
```

After the cloning, run
```sh
composer install
```
```sh
npm install
```

```sh
php artisan migrate
```

```sh
php artisan db:seed
```
Once the seeding is done, the users and products are created. 

You can now run
```sh
php artisan serve --port=8000
```
